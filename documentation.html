jQuery Fleeckr

Flickr API plugin

Available methods

Currently it only accept the getPhotos methods that don't require any kind of permission. Every method requires an extra parameter.

flickr.galleries.getPhotos
Get photos from a gallery.
extra syntax: gallery_id=([0-9]-[0-9])
e.g.: 6065-72157617483228192
check this out
flickr.groups.pools.getPhotos
Get photos from a group's pool.
extra syntax: group_id=([0-9]@N[0-9])
e.g.: 86881528@N00
flickr.panda.getPhotos
Get photos from asking a Panda
extra syntax: panda_name=(ling ling|hsing hsing|wang wang)
flickr.people.getPublicPhotos
Get someone's uploaded photos
extra syntax: user_id=([0-9]@N[0-9])
e.g.: 31579803@N08
flickr.people.getPhotosOf
Get photos in which someone's is tagged
extra syntax: user_id=([0-9]@N[0-9])
e.g.: 31579803@N08 for tomasdev
flickr.photosets.getPhotos
Get photos from a set
extra syntax: photoset_id=([0-9])
e.g.: 72157623977103845

Available templating tags

{{title}} : name of image
{{image_s}} : image is size s (75x75)
{{image_m}} : image is size m (medium size, about 300x200)
{{image_b}} : image is size b (big size, about 1024x768)
{{title}} : name of image

General options

  • flickr: flickr API URL. Default: 
    'http://api.flickr.com/services/'
  • limit: quantity of photos to append. Default: 
    20
  • apikey: (required) API Key got from registering an API key. Default: 
    none
  • jsoncallback: callback after getJSON is got an answer. Default: 
    '?'
    (fleeckr function)
  • method: one of Available method above mentioned. Default: 
    'flickr.panda.getPhotos'
  • itemTemplate: markup for every photo result using Available templating tags mentioned above. Default: 
    '<div class="fleeckr-image"><a href="{{image_b}}" title="{{title}}"><img alt="{{title}}" src="{{image_s}}" /></div>'
  • extra: (required when method given) the extra parameter in the syntax need documented in this page. Default: 
    'panda_name=wang wang'
  • itemCallback: You can add a callback on each item. The scope is set to the container and the item object is made available. Default: 
    none

Instructions

1 - Please register for an API key in Flickr.
2 - Include jFleeckr script
3 - On document ready, assign jFleeckr to an element. See example.
4 - Modify fleeckr.css as need.
5 - Enjoy.
